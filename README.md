## 📖 Sobre o Projeto

O **Refund** é uma aplicação web de controle de despesas e reembolsos desenvolvida do zero com HTML, CSS e JavaScript puro. O usuário informa os dados da despesa — nome, categoria e valor — e a solicitação é registrada em um painel ao lado, exibindo o total acumulado. As despesas são analisadas e reembolsadas em até **30 dias**.

## ✨ Funcionalidades

- 📝 **Formulário de solicitação** — informe nome, categoria e valor da despesa com facilidade
- 📋 **Painel de solicitações** — visualize todas as despesas adicionadas em tempo real
- 💰 **Total acumulado** — acompanhe o valor total das solicitações do mês
- 🗂️ **Categorias** — organize suas despesas por tipo para melhor controle
- ⚡ **100% no front-end** — sem necessidade de back-end ou banco de dados

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura e semântica da aplicação
- **CSS3** — estilização, layout responsivo e identidade visual
- **JavaScript** — lógica de negócio, manipulação do DOM e interatividade
